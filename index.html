Portal de Juegos ESIGames

esiGames

En nuestro portal podrás jugar minijuegos como el FlappyBird, Tetris, Pong, Snake y BrickBreaker.
Implementado con Python haciendo uso de las librerías Pygame y Tkinter.
Proyecto realizado en colaboración con Jose Luis Nieto y Johnatan Mora.

Reconocedor de matriculas

Reconocedor de Matriculas

Implementado con Python haciendo uso de las librerías OpenCV y OCR.
Página web en la que podrás cargar una imagen de un coche y se identificara su matricula.
Proyecto realizado en colaboración con Jose Luis Nieto.

BetGenius

BetGenius

Bot de Telegram que predice el resultado de los partidos de futbol que se juegan a diario.
Implementado en Python haciendo uso de la API de BotTelegram y APIFootball. También consta de sitio web para el registro y acceso al bot.
Proyecto realizado en colaboración con Jose Luis Nieto y Johnatan Mora.

Aplicacion de escritorio para ver información sobre Pokemons y realizar combates entre ellos.

Proyecto realizado en colaboración con Jose Luis Nieto.
